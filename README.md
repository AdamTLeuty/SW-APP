# Smile Correct Club Patient Portal

## Tech Stack
### Client
- React Native / Expo
-- Axios (For connecting to the server)

### Server
- Golang
-- Gin (As the web framework)
- Sqlite (For storing user login details, and references to UGC images)
- BCrypt (For encrypting user passwords)
